# Little Explorer Learning World — V7 LTS 1.0 Stable

This is the cleaned long-term-support baseline built from V7.0.1.

## Run
Open `index.html` in Chrome, or deploy the folder contents to the root of GitHub Pages.

## Included
- Dashboard
- Weekly Planner
- Reading Journey
- Continue Reading
- Vocabulary cards
- Sentence Bank
- Bookshelf
- Data import/export
- Existing PDF preview workflow

## Data
Application data is stored locally in the browser. Export a backup before replacing an older deployment.

Reading Journey add-on: Close Reading Guide 2.1 Fixed (P3 Standard + Challenge), based on the supplied Wonder challenge and Unit 1 workbook.
