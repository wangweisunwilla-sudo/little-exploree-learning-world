Little Explorer Learning World V18 Family Sync · Official GIS

上传方法：
1. 解压 ZIP。
2. 将 index.html、assets 文件夹和本说明上传到 GitHub 仓库根目录并覆盖旧文件。
3. 等待 GitHub Pages 部署完成。
4. 用 HTTPS 网站打开，并按 Command+Shift+R 强制刷新。

Google Cloud 必需配置：
- OAuth Client 类型：Web application
- Authorized JavaScript origin：https://wangweisunvilla-sudo.github.io
- Google Drive API：Enabled
- Data Access scope：https://www.googleapis.com/auth/drive.file
- 测试期间：登录账号加入 Test users

首次使用：
1. 浏览器允许该网站弹出窗口。
2. 点击“连接 Google Drive”。
3. 完成 Google 账号选择和授权。
4. 点击“保存到云端”。
5. 另一台设备先连接，再点击“从云端载入”。

安全说明：
- 前端只包含 OAuth Client ID，不包含 Client Secret。
- 使用 drive.file 最小权限。
