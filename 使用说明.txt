Little Explorer Learning World V17.1 Family Sync Stable

上传到 GitHub
1. 解压 ZIP。
2. 打开 GitHub 仓库 little-exploree-learning-world。
3. Add file → Upload files。
4. 上传 index.html、assets 文件夹和本说明文件。
5. 点击 Commit changes。
6. 等待 Actions 显示绿色勾，再从 GitHub Pages 网站打开。

正确网站地址
https://wangweisunvilla-sudo.github.io/little-exploree-learning-world/

重要：不能从 Finder 双击 index.html。地址栏出现 file:// 时，Google OAuth 一定会失败。

Google Cloud 检查
- OAuth Client 类型：Web application
- Authorized JavaScript origins：
  https://wangweisunvilla-sudo.github.io
  不带仓库路径，不带末尾斜杠。
- Authorized redirect URIs：留空
- Audience → Test users：加入实际登录的 Gmail
- APIs & Services：启用 Google Drive API

网页只使用 OAuth Client ID，不包含 Client Secret。
