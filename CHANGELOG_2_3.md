# Reading Guide 2.3

## Vocabulary card fix
- Automatically merges exact duplicate vocabulary cards within the same book/chapter context.
- Cleans duplicates already stored in browser localStorage when the page loads.
- Editing a chapter now removes stale word cards that were deleted from the chapter form.
- Saving the same manual word card again updates the existing card instead of creating another copy.
- Keeps the same word as separate cards when it belongs to different chapters, so context is not lost.

## Compatibility
- Uses the same localStorage key and data structure.
- Existing books, chapter records, quotes, progress and weekly plans remain available.
