# Reading Journey Guide Upgrade 2.1

This build keeps the V7 LTS stable core and upgrades only the Reading Journey guidance and chapter-record workflow.

## What changed

- Five-step close-reading routine: **Read → Think → Notice → Explain → Write**.
- Three task levels: **CORE**, **TALK**, and **CHALLENGE**.
- A Wonder Unit 1 map for Chapters 1–5, with chapter-specific reading skills, questions, vocabulary direction, sentence patterns, writing outcomes, and read-aloud coaching.
- Reusable skill tools: Vocabulary Hunter, Grammar Detective, Show, Don’t Tell, and Precept & Theme.
- Parent/teacher prompts and attention-based stop rules.
- New chapter-record fields: prediction, one-sentence summary, evidence, reading skill, task level, and optional challenge response.
- Leo Reading Coach now rewards evidence-based thinking with an **Evidence Explorer** badge.

## Compatibility

Existing browser data, chapter records, vocabulary cards, sentence bank entries, weekly plans, JSON backups, and Google Drive sync data remain compatible. New fields are optional and are added only when a new or existing chapter record is saved.

## Recommended use for P3

Complete the CORE task first. Use TALK when writing is difficult. Add CHALLENGE only when the core answer is already clear and supported by evidence.
